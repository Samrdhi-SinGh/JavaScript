String :-

A string represents text with single quotes otr double quotes. Like: 'Hello', 'Quards'.

Syntax rules for a String :-
For a String, we right Text in between single quotes or double quotes.

Concatenation :-
When we combine two data's together, it's Concatenation.
Like: Combining two strings, Two integers, or one String and one integer together.
Example: 'Str'+'ing' = 'String', 1+2 = 3

Note: '+' when in between same values like int, string will always add it, and when between string and int will always concatenate.

Type Coercion (Automatic type conversion) :-
If we add 'int' and 'string' together, 'int' will convert into 'string' automatically.
Example: 'Hello'+3 = 'Hello3' (A string)

typeof :- typeof is an instruction used to tell the value of a data.
Like: typeof(2)- number, typeof('hello')- String.

JavaScript calculates from left to right only.
The output of JavaScript will always come in double quote.

3 ways to write string:-

1. '' - Single quote
2. "" - Double quote
3. `` - Back tick

Character:-
1. Letter
2. Number
3. Symbol
4. Escape character

Escape character: use to print thing and give spaces, next line and many more.

\' - will print ' in output
\" - print " in output 
\n - new line escape Character

Mostly '' should be used because it's easy and simple.

But to print ' we use double quote or escape character.

Strings created with back ticks(``) have a special name called 'Template strings'.
We call them this because they have special features.

1. Interpolation :-  
${} = Allows to insert value directly into string.
Interpolation is a much cleaner of inserting value into a string.

Example: `Item(${(1+1)}): $${28.94}`

2. Multi-line strings :-
It is used to print two line into one without using anything.
Like; 
`some
text`
= 'some\ntext'

Whenever we want to insert a value into a string, we use 'Template string'.
